Use this webpage to search global emission factors available through the Climatiq API

Climatiq API converts operational activity data into CO2e (carbon dioxide equivalent) estimates, allowing us to understand each activity's impact on the environment and identify actions to manage and reduce it.

What is an Emission Factor? An emission factor is a value that estimates the quantity of a gas with a global warming potential (GWP) released into the atmosphere, per some unit related to a particular activity.

Make use of the filters and search bars to toggle around the different activities and get to know each emission factor. You can find and group the variety of activities using parameters such as:
