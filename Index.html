This website is developed as part of Project 2 of the Code Academy Berlin. It pulls live data from Climatiq API displaying global emission factors worldwide.

Climatiq API converts operational activity data into CO2e (carbon dioxide equivalent) estimates, allowing to calculate each activity's impact on the environment.

What is an Emission Factor? An emission factor is a value that estimates the quantity of a gas with a global warming potential (GWP) released into the atmosphere, per some unit related to a particular activity.
